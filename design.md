# 数学小天才 - 移动应用界面设计

## 设计理念

本应用专为3-6岁儿童设计，采用**动物和水果主题**，界面色彩鲜艳、角色可爱。遵循 Apple Human Interface Guidelines (HIG)，确保应用在 iOS 上感觉像原生应用。

## 主要内容和功能

### 首页 (Home)

**布局设计：**
- 顶部：应用标题 + 设置按钮
- 中部：四个大型模块卡片（2x2网格）
  - 基础认知（星星图标，黄色）
  - 计数能力（数字图标，蓝色）
  - 运算启蒙（加号图标，绿色）
  - 生活应用（时钟图标，粉色）
- 底部：成就按钮 + 学习进度条

**交互：**
- 点击卡片进入对应模块
- 卡片有轻微弹跳动画
- 背景播放轻快背景音乐

### 模块详情页

**布局设计：**
- 顶部：返回按钮 + 模块标题
- 中部：子模块卡片列表（纵向滚动）
- 每个卡片显示：
  - 可爱的图标/插图
  - 子模块名称
  - 完成进度（星星数）

### 游戏屏幕通用布局

**布局设计：**
- 顶部：返回按钮 + 关卡标题 + 进度指示
- 中部：主游戏区域（占屏幕70%）
- 底部：操作区域/选项区域

**反馈系统：**
- 正确：绿色对勾 + 烟花动画 + "真棒！"语音
- 错误：红色叉号抖动 + "再试试！"语音
- 完成：气球升起 + 获得勋章动画

## 关键用户流程

### 流程1：数字学习
1. 首页 → 点击"基础认知"卡片
2. 基础认知页 → 点击"认识数字"
3. 数字动物乐园 → 选择数字0-10
4. 查看数字动画 → 听读音和动物叫声
5. 完成互动 → 获得星星奖励

### 流程2：加法练习
1. 首页 → 点击"运算启蒙"卡片
2. 运算启蒙页 → 点击"加法启蒙"
3. 选择游戏类型（蝴蝶/水果/青蛙）
4. 完成加法题目 → 查看动画演示
5. 答对 → 庆祝动画 + 获得勋章

### 流程3：时间认识
1. 首页 → 点击"生活应用"卡片
2. 生活应用页 → 点击"认识时间"
3. 选择整点/半点/排序
4. 互动学习时钟 → 听报时语音
5. 完成练习 → 获得贴纸奖励

## 字体规范

- 标题：圆体/系统默认，36pt，加粗
- 子标题：圆体/系统默认，24pt，中粗
- 正文：系统默认，18pt，常规
- 数字：等宽字体，大号显示

## 动画规范

| 场景 | 动画类型 | 时长 |
|-----|---------|------|
| 按钮点击 | 缩放0.95 | 100ms |
| 页面切换 | 滑入 | 300ms |
| 正确反馈 | 弹跳+烟花 | 500ms |
| 错误反馈 | 抖动 | 300ms |
| 数字动物 | 特定动作 | 1000ms |
| 庆祝动画 | 气球升起 | 2000ms |

## 音效规范

| 场景 | 音效类型 |
|-----|---------|
| 按钮点击 | 轻柔"咔嗒"声 |

## 无障碍设计

- 所有按钮足够大（最小44x44pt）
- 色彩对比度符合WCAG标准
- 支持VoiceOver
- 动画可关闭选项
